# Сервис получения событий (Evnet Receiver)

Сервис, получающий события из внешних источников и перенаправляющий их на обработку

[Описание поротокола](https://github.com/dws-1-2026-green/wiki/blob/api/docs/api/api.md#сервис-получения-событий)

# Настройки

Настройки происходят через переменные среды.

Также поддерживает .env файл

### LOG_LEVEL
Уровено логирования
- `debug`
- `info`
- `warn`
- `error`

По умолчанию `info`

### LOG_FORMAT
Определяет формат выводимых логов
- `json` - выводит логи в виде json объектов
- `plain` - выводит логи в виде текста

По умолчанию `plain`

### IP
Слушаемый ip адрес http сервера

По умолчанию `0.0.0.0`

### PORT
Используемый http сервером порт

По умолчанию `8080`

### KAFKA_BROKERS
Список kafka брокеров, через запятую

По умолчанию `localhost:9092`

### KAFKA_TOPIC
Топик, в который отправляются события

По умолчанию `routing.requests`
